# Headers
Displays information to users. Sometimes interactive, sometimes not.
Some of them feel like they should be "Cards"...

# Data Header

##### Used to display data to users. Very much designed to display staking data.

## Props
- **`primaryLabel`** - <RequiredText /> - `string` - The Label for the top information ("My Delegations" in this example).
- **`primaryValue`** - <RequiredText /> - `string` - The Value below the top label ("$2,899.28" in this example).
- **`secondaryLabel`** - `string` - The first Label on the "second row" ("Available Rewards" in this example).
- **`secondaryValue`** - `string` - The first Value on the "second row" ("$33.84" in this example).
- **`tertiaryLabel`** - `string` - The second Label on the "second row" ("Undelegations" in this example).
- **`tertiaryValue`** - `string` - The second Value on the "second row" ("0" in this example).
- **`actionButtons`** - `ReactNode` - Buttons for users to do things.

## Import
```jsx
import { DataHeader } from "station-ui";
```

# Rich Header

##### A Header for displaying a large image, title, description, and a link.

## Props
- **`image`** - <RequiredText /> - `string` - The image to display.
- **`title`** - <RequiredText /> - `string` - The title of the header.
- **`link`** - <RequiredText /> - `string` - The external link to display.
- **`description`** - `string` - The description of the header that is full width.
- **`status`** - `string` - Adds an icon to the image for more information to give to the user.
  - Options are `alert`, `check`.

# Section Header

##### A small header to display a label.

## Props
- **`title`** - `string` - The title of the header.
- **`extra`** - `ReactNode` - Adds whatever is passed on the far ride side of the header.
- **`withLine`** - `string` - Adds a line to the header. (Makes the label middle aligned).
- **`indented`** - `string` - Adds a small padding to the left and right of the header.
- **`icon`** - `string` - Adds an icon in front of the label (the prop "title").
- **`className`** - `string` - Adds a class to the header.

import { SectionHeader } from "station-ui";
