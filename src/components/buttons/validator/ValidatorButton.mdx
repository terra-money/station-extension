# ValidatorButton
Button for validator.

## Import
```jsx
import { ValidatorButton } from '@terra-station/components/buttons';
```

# ValidatorButton Props
* **`validatorLabel`** - <RequiredText /> - `string` - Main label of the ValidatorButton.
* **`validatorSubLabel`** - <RequiredText /> - `string` - Sub label that is below the main label.
* **`imgSrc`** - <RequiredText /> - `string` - Source of the Validator image.
