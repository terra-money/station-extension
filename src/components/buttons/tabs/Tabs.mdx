# Tabs

This is used for filtering and switching between different views.

## Import

```jsx
import { Tabs } from "@terra-money/station-ui/components/buttons";
```

# Tabs Props

- **`tabs`** - <RequiredText /> - Used to set the tabs.
- - **`tabs.key`** - `string` - For identifying tab.
- - **`tabs.label`** - `string` - The label for the tab.
- - **`tabs.onClick`** - `() => void` - Called when associated tab is clicked.
- **`activeTabKey`** - `string` - The key of the active tab.
