# Navigational Buttons
These are buttons that are used for navigational purposes.
* **`NavButton`** - Likely only used for navigating through settings pages or page routing.
* **`RoundedActionButton`** - This button is used for the mobile bottom navigation bar.

## Import

```jsx
import { NavButton, RoundedActionButton } from '@terra-station/components/buttons';
```

# NavButton Props
##### The `NavButton` Used for navigating settings or page routing.
* **`label`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>*required*</span> - Sets the label for the button.
* **`subLabel`** - Sets right side 'sub'-label.
* **`icon`** - Sets icon that is left of the `label`.
