# Button Wrapper

- **`ButtonInlineWrapper`** - Used to wrap multiple call to actions.

## Import

```jsx
import {
  SubmitButton,
  ButtonInlineWrapper,
} from "@terra-money/station-ui/components/buttons";
```

# ButtonInlineWrapper

##### A container for multiple submit buttons.

- **`buttons`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>_required_</span> - An array of buttons to be wrapped.
  - **Should typically be a "Cancel" or "Back" button**
