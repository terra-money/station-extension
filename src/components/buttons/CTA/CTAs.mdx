# Call To Action Buttons (CTAs)
These buttons are used for a "call to action".
They are used to direct the user to a specific page or action.
We have 2 main types:
* **`SingleCTA`** - Used for a single call to action.
* **`SequentialCTAs`** - Used for 2 call to actions.

## Import

```jsx
import { SingleCTA, SequentialCTAs } from '@terra-station/components/buttons';
```

# SingleCTA Props
##### The `SingleCTA` button is used for either user form submitting or routing.

* **`color`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>*required*</span> - Sets the color of the button.
  * Options are `primary`, `secondary`, `destructive`.
* **`label`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>*required*</span> - Sets the label of the button.
* **`loading`** - Sets the button to a loading state (replaces icon while loading).
* **`icon`** - Appends an icon to the button. In front of the label.

> ##### `<SingleCTA />` is a full width button.
> - Built on top of `<Button />`.
> - Has a `type='submit'` attribute by default.
> - `block` by default.
> - Is disabled when `loading` is active.

# SequentialCTAs
##### A container for 2 CTA buttons.
* **`firstCTA`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>*required*</span> - Left CTA in the container.
  * **Should typically be a "Cancel" or "Back" button**
* **`secondCTA`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>*required*</span> - Sets the label of the button.
  * **Should typically be a "Submit" or "Next" button**
