# Submit Button

This button is used for form submission or "call to action".

## Import

```jsx
import { SubmitButton } from "@terra-money/station-ui/components/buttons";
```

# SubmitButton Props

##### The `SubmitButton` button is used for either user form submitting or routing.

- **`variant`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>_required_</span> - Sets the variant of the button.
  - Options are `primary`, `secondary`, `destructive`.
- **`label`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>_required_</span> - Sets the label of the button.
- **`loading`** - Sets the button to a loading state (replaces icon while loading).
- **`icon`** - Appends an icon to the button. In front of the label.

> ##### `<SubmitButton />` is a full width button.
>
> - Built on top of `<Button />`.
> - Has a `type='submit'` attribute by default.
> - `block` by default.
> - Is disabled when `loading` is active.
