# Base Buttons

These are the Base buttons accessed through `<Button />`. It is used for all actions on the page. It has a few different variations that can be used depending on the action.

## Import

```jsx
import { Button } from "@terra-station/components/buttons";
```

## Props

##### Used as the base for most buttons.

- **`variant`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>_required_</span> - Sets the variant of the button.
  - Options are `primary`, `secondary`, `destructive`, `dashed`, `white-filled`, `outlined`.
- **`label`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>_required_</span> - Sets the label of the button.
- **`loading`** - Sets the button to a loading state (replaces icon while loading).
- **`block`** - Sets the button to take up the full width of its container.
- **`disabled`** - (included in `ButtonHTMLAttributes`) Sets the button to a disabled state.
