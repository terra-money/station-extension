# Checked Button

This button is used for toggling a state. It is used in the `Create New Wallet` page for confirming seed phrase toggling .

## Import

```jsx
import { CheckedButton } from "@terra-money/station-ui/components/buttons";
```

# CheckedButton Props

##### The `CheckedButton` is build on top of `<Button />`.

- **`label`** - <span style={{ color: "var(--token-error-500)", fontWeight: "bold", fontSize: "14px" }}>_required_</span> - Sets the label for the button.
- **`active`** - Sets the button to be active.
- **`onClick`** - Sets the onClick handler for the button. Will likely be a function to set the button to an active state.
