# Default Empty UI
Displays an "empty" icon and "No results found".

## Props
- `size` - Passes in a size to the mui icon.
- `thickness` - Passes in a thickness to the mui icon.

## Imports
```tsx
  import { LoadingCircular } from '@terra-station-ui/components/feedback';
```
