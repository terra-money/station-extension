# Page Tabs

##### These tabs are used to navigate between pages.

## Props
- **`tabs`** - <RequiredText /> - `string[]` - The label of the tabs.
- **`activeTab`** - <RequiredText /> - `number` - The index of the active tab.
- **`onClick`** - <RequiredText /> - `(index: number) => void` - Controls what happens on click.

## Import
```jsx
import { PageTabs } from "station-ui";
```
