# Radio List

##### This is a list of radio buttons.

## Radio Full List Props
- **`children`** - <RequiredText /> - `ReactNode` - The content of the list. Should be `<RadioListItem />`'s.

## Radio List Item Props
- **`label`** - <RequiredText /> - `string` - The label of the radio button.
- **`checked`** - `boolean` - Sets the radio button to checked.
- **`onClick`** - <RequiredText /> - `() => void` - Controls what happens on click.
- **`isOpen`** - `boolean` - Tells if the accordion is open or closed.
- **`setOpenAcc`** - `any` - Sets the accordion to open or closed.
- **`accContent`** - `[{ name: string, icon: string }]` - Sets the content of the accordion.

## Import
```jsx
import { RadioList, RadioListItem } from "station-ui";
```
