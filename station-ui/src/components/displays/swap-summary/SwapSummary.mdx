# Swap Summary

##### This just displays a summary of a swap.

## Swap Summary Props
- **`fromAsset`** - <RequiredText /> - `**below object**` - The from asset information.
  - `{`
    `symbol: string`
    `amount: string`
    `icon: string`
  `}`
- **`toAsset`** - <RequiredText /> - `**below object**` - The to asset information.
  - `{`
    `symbol: string`
    `amount: string`
    `icon: string`
  `}`

## Import
```jsx
import { SwapSummary } from "station-ui";
```
