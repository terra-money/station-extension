# Voting Summary

##### Displays more information on top of `<ProgressBar />`.

## VotingSummary Props
- **`data`** - <RequiredText /> - `ProgressBarData[]` - The status of each step.
  - **`ProgressBarData`** = `{`
    `type: 'yes' | 'abstain' | 'no' | 'noWithVeto'`
    `percent: string`
    `amount: string | ReactNode`
  `}`
- **`threshold`** - `number` - The threshold to pass.

## Import
```jsx
import { VotingSummary } from "station-ui";
```
