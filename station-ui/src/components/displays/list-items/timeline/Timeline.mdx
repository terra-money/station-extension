# Timeline

##### Timeline component to show details of a transaction.

## Props
- **`startOverride`** - `ReactNode` - Overrides the start item.
  - ***The only component passed should be `<ActivityListItem />`***
- **`startItem`** - `**object below**` - The first item of the timeline. It is different from the middle items.
  - `{`
    - **chain:** `{ icon: string, label: string },`
    - **coin:** `{ icon: string, label: string },`
    - **msg:** `ReactNode`
  - `}`
- **`middleItems`** - `**object below**` - All middle items of the timeline. **Note: if one is passed in the line is not displayed**
  - `{`
    - **variant:** `"default" | "success" | "warning",`
    - **msg:** `ReactNode,`
    - **warningPillText?:** `string`
    - **transactionButton?:** `{`
      - **label:** `string`
      - **onClick:*** `() => void`
    - `}`
    - **disabled?:** `boolean`
  - `}[]`
- **`endItem`** - `**object below**` - The last item of the timeline.
  - `{`
    - **chain:** `{ icon: string, label: string },`
    - **coin:** `{ icon: string, label: string },`
    - **msg:** `ReactNode`
  - `}`
- **`forceShowAll`** - `boolean` - Forces the timeline to show all items.

## Import
```jsx
import { Timeline } from "station-ui";
```
