# Token ListItem

##### Displays a single token item.

## Props
- **`balance`** - `string` - The balance of the token.
  - If this is not provided, the balance will default to "0".
- **`chains`** - <RequiredText /> - `string[]` - A list of chains this token is on.
- **`currency`** - <RequiredText /> - `**object below**` - The currency a user uses.
  - `{ id: string, symbol: string, name: string }`
- **`tokenImg`** - <RequiredText /> - `string` - The Image for the token.
- **`symbol`** - <RequiredText /> - `string` - The label to display the token.
- **`price`** - `number` - The current price of the token.
- **`change`** - `number` - The percent change of the token in 24 hours.
- **`amountNode`** - <RequiredText /> - `ReactNode` - The amount of the token.
- **`onClick`** - <RequiredText /> - `() => void` - Function that is called when the item is clicked.

## Import
```jsx
import { TokenListItem } from "station-ui";
```

## Stories

## Default
Displays an individual token item with a Tooltip that displays amounts on chains (if token is on multiple chains).

# TokenSingleChain ListItem

##### Displays a single token what has a single chain.

## Props
- **`balance`** - `string` - The balance of the token.
  - If this is not provided, the balance will default to "0".
- **`chain`** - <RequiredText /> - `**object below**` - A list of chains this token is on.
  - `{ icon: string, label: string }`
- **`currency`** - <RequiredText /> - `**object below**` - The currency a user uses.
  - `{ id: string, symbol: string, name: string }`
- **`tokenImg`** - <RequiredText /> - `string` - The Image for the token.
- **`symbol`** - <RequiredText /> - `string` - The label to display the token.
- **`price`** - `number` - The current price of the token.
- **`isSendBack`** - `boolean` - If the token is being sent back.
- **`amountNode`** - <RequiredText /> - `ReactNode` - The amount of the token.
- **`onClick`** - <RequiredText /> - `() => void` - Function that is called when the item is clicked.

## Import
```jsx
import { TokenSingleChainListItem } from "station-ui";
```

## Stories

## Chain Display
A token displayed with a chain focus. So, removed 24hour change and replaces it with a chain icon and name.
<div className="story_with_source_below">
  <Canvas of={TokenListItemStories.ChainDisplay} />
  <Source code={`

  tokenImg={"https://station-assets.terra.dev/img/coins/Luna.svg"}
  symbol={"LUNA"}
  price={1}
  amountNode={<span>420.00</span>}
  chain={{ icon: "https://station-assets.terra.dev/img/chains/Terra.svg", label: "Terra" }}
/>
  `} />
</div>

## Send Back
The same as ChainDisplay, but with a send back pill.

  <Canvas of={TokenListItemStories.ChainDisplay} />

# TokenCheckbox ListItem

##### Displays a token item with a checkbox.

## Props
- **`tokenImg`** - <RequiredText /> - `string` - The Image for the token.
- **`symbol`** - <RequiredText /> - `string` - The label to display the token.
- **`chain`** - <RequiredText /> - `**object below**` - A list of chains this token is on.
  - `{ icon: string, label: string }`
- **`onClick`** - <RequiredText /> - `() => void` - Function that is called when the item is clicked.
- **`checked`** - `boolean` - If the checkbox is checked.

## Import
```jsx
import { TokenCheckboxListItem } from "station-ui";
```

## Stories

## With Chain Checkbox
The same as ChainDisplay, but with a checkbox on the right side instead of the amounts.

  checked={checked}
  chain={{ icon: "https://station-assets.terra.dev/img/chains/Terra.svg", label: "Terra" }}
/>
  `} />
</div>
