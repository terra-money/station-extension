# AddressSelectable ListItem

##### A Selectable List Item that displays an address.

## Props
- **`label`** - <RequiredText /> - `string` - The largest text (in this example is "Terra").
- **`subLabel`** - <RequiredText /> - `string` - Smaller text below label.
- **`active`** - `boolean` - Add an "active" pill next to the label.
- **`chain`** - `**object below**` - Chain for image.
  - `{ icon: string, label: string }`
- **`onClick`** - <RequiredText /> - `() => void` - Function that is called when the item is clicked.

## Import
```jsx
import { AddressSelectableListItem } from "station-ui";
```

# WalletSelectable ListItem

##### A Selectable List Item that displays a Wallet.

## Props
- **`label`** - <RequiredText /> - `string` - The largest text (in this example is "Terra").
- **`subLabel`** - <RequiredText /> - `string` - Smaller text below label.
- **`active`** - `boolean` - Add an "active" pill next to the label.
- **`copyValue`** - <RequiredText /> - `string` - String that is copied to a users clipboard when `<Copy />` is clicked.
- **`onClick`** - <RequiredText /> - `() => void` - Function that is called when the item is clicked.
- **`settingsOnClick`** - <RequiredText /> - `() => void` - Function that is called when the settings icon is clicked.

## Import
```jsx
import { WalletSelectableListItem } from "station-ui";
```
