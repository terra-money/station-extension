# Wallet List Item

##### Display a wallets icon, name, and address.

## Props
- **`name`** - <RequiredText /> - `string` - The name of the wallet.
- **`address`** - <RequiredText /> - `string` - The address of the wallet.
- **`emoji`** - <RequiredText /> - `string` - The user input Emoji or Letter.
- **`walletBalance`** - `string` - When passed in, it displays the balance on the right side of the list item.

## Import
```jsx
import { WalletListItem } from "station-ui";
```
