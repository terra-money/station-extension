# Wallet List

##### Displays wallets. The wallet list item is `<WalletSelectableListItem />`.

## WalletList Props
- **`activeWallet`**: - <RequiredText /> - `**object below**` - Data of the active wallet.
  - `{`
    - `name: string;`
    `address: string;`
    `settingsOnClick: () => void;`
  - `}`
- **`otherWallets`**: - <RequiredText /> - `**object below**` - Array containing data for other wallets.
  - `{`
    - `name: string;`
    `address: string;`
    `onClick: () => void;`
    `settingsOnClick: () => void;`
  - `}[]`

## Import
```jsx
import { WalletList } from "station-ui";
```
