# Progress Tracker

##### This displays the progress of a transactions.

## ProgressTracker Props
- **`steps`** - <RequiredText /> - `StepStatus[]` - The status of each step.
  - **`StepStatus`** = `'incomplete' | 'completed' | 'inProgress' | 'failed'`
- **`stepLabels`** - `string[]` - Adds labels below the steps.

## Import
```jsx
import { ProgressTracker } from "station-ui";
```
