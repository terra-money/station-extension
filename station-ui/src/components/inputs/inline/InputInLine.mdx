# InputInLine

##### An inpit with a label inline (inside) with the input.

## Props
- **`label`** - <RequiredText /> - `string` - The label next to the checkbox.
- **`extra`** - `ReactNode` - Adds anything on the right side of the input.

## Import
```jsx
import { InputInLine } from "station-ui";
```

## Stories

## Example With Extra Paste
Label in the input with a placeholder and ReactNode passed in as 'extra' on the right side.
<div className="story_with_source_below">
  <Canvas of={InputInLineStories.ExampleWithExtraPaste} />
  <Source code={`
    const [value, setValue] = useState('');

return (
  <InputInLine
    label={'To'}
    extra={
      <Paste
        onPaste={(text) => setValue(text)}
      />
    }
    value={value}
    onChange={(e) => setValue(e.target.value)}
    placeholder='Primary input'
  />
)
  `} />
</div>
