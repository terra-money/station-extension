# Checkbox

##### This is a checkbox component.

## Props
- **`label`** - <RequiredText /> - `string` - The label next to the checkbox.
- **`checked`** - `boolean` - Displays a filled checkbox if true.

## Import
```jsx
import { Checkbox } from "station-ui";
```
