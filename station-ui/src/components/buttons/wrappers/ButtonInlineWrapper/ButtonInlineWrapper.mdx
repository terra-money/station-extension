# Button Wrapper

- **`ButtonInlineWrapper`** - Used to wrap multiple call to actions.

## Import

```jsx
import {
  SubmitButton,
  ButtonInlineWrapper,
} from "@terra-money/station-ui/components/buttons";
```

# ButtonInlineWrapper

##### A container for multiple submit buttons.

- **`buttons`** - <RequiredText /> - An array of buttons to be wrapped.
  - **Should typically be a "Cancel" or "Back" button**
