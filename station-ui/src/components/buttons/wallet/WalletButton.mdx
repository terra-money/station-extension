# WalletButton

##### Selector button for wallets.

## Props
- **`emoji`** - **`string`** - An emoji that is passed in.
- **`walletName`** - <RequiredText /> - **`string`** - The name of the wallet.
- **`walletAddress`** - <RequiredText /> - **`string`** - The address of the wallet.
- **`chainIcon`** - <RequiredText /> - **`string`** - The chain icon of the wallet.
- **`onClick`** - <RequiredText /> - **`() => void`** - Onclick functionality for the button.

## Import
```jsx
import { WalletButton } from "station-ui";
```
