# Buttons

These are the Base buttons accessed through `<Button />`. It is used for all actions on the page. It has a few different variations that can be used depending on the action.

## Import

```jsx
import { Button } from "@terra-money/station-ui/components/buttons";
```

## Base [<span style={{ fontSize: "14px", fontWeight: "600", marginLeft: "24px" }}>Docs<RightArrow style={{ fill: "black" }} /></span>](/docs/components-buttons-base--docs)

##### Used as the base for most buttons.

- **`variant`** - <RequiredText /> - Sets the variant of the button.
  - Options are `primary`, `secondary`, `warning`, `dashed`, `white-filled`, `outlined`.
- **`label`** - <RequiredText /> - `string` - Sets the label of the button.
- **`loading`** - `boolean` - Sets the button to a loading state (replaces icon while loading).
- **`block`** - `boolean` - Sets the button to take up the full width of its container.
- **`icon`** - `ReactNode` - Appends an icon to the button. In front of the label.
- **`disabled`** - (included in `ButtonHTMLAttributes`) Sets the button to a disabled state.

# Submit Button

This button is used to direct the user to a specific page or action.
We have 2 main types:

- **`SubmitButton`** - Used for a single call to action.
- **`ButtonInlineWrapper`** - Used for 2 call to actions.

## Import

```jsx
import {
  SubmitButton,
  ButtonInlineWrapper,
} from "@terra-money/station-ui/components/buttons";
```

## SubmitButton [<span style={{ fontSize: "14px", fontWeight: "600", marginLeft: "24px" }}>Docs<RightArrow style={{ fill: "black" }} /></span>](/docs/components-buttons-submit--docs#SubmitButton-props)

##### One button for submitting a form or directing the user to a specific page.

- **`variant`** - <RequiredText /> - Sets the variant of the button.
  - Options are `primary`, `secondary`, `warning`.
- **`label`** - <RequiredText /> - `string` - Sets the label of the button.
- **`loading`** - `boolean` - Sets the button to a loading state (replaces icon while loading).
- **`icon`** - `ReactNode` - Appends an icon to the button. In front of the label.

> ##### `<SubmitButton />` is a full width button.

# Rounded

This is a round button.

- **`RoundedButton`** - Used in Portfolio section actions.
- **`FlipButton`** - Very specifically used in Swap to switch the To & From tokens.

## Import

```jsx
import {
  RoundedButton,
  FlipButton,
} from "@terra-money/station-ui/components/buttons";
```

## RoundedButton [<span style={{ fontSize: "14px", fontWeight: "600", marginLeft: "24px" }}>Docs<RightArrow style={{ fill: "black" }} /></span>](/docs/components-buttons-rounded--docs#rounded)

##### The `RoundedButton` Used in Portfolio section actions.

##### Used for routing and not submitting.

- **`color`** - <RequiredText /> - Sets the color of the button.
  - Options are `primary`, `secondary`.
- **`icon`** - <RequiredText /> - `ReactNode` - Sets the icon of the button.
- **`size`** - Sets the size of the button.
  - Options are `default`(50px, 50px), `small` (42px, 42px), `large` (64px, 64px).

# Navigational Buttons

- **`NavButton`** - Likely only used for navigating through settings pages or page routing.
- **`RoundedActionButton`** - This button is used for the mobile bottom navigation bar.

## Import

```jsx
import {
  NavButton,
  RoundedActionButton,
} from "@terra-money/station-ui/components/buttons";
```

## NavButton [<span style={{ fontSize: "14px", fontWeight: "600", marginLeft: "24px" }}>Docs<RightArrow style={{ fill: "black" }} /></span>](/docs/components-buttons-navigation--docs#navbutton-props)

##### The `NavButton` Used for navigating settings or page routing.

- **`label`** - <RequiredText /> - `string` - Sets the label for the button.
