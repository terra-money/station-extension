# Summary Header

##### A Summary Header to show information. With a large status icon.

## Props
- **`statusLabel`** - <RequiredText /> - `string` - Largest text.
- **`statusMessage`** - <RequiredText /> - `string` - Secondary text.
- **`status`** - `string` - Determines the icon.
  - Options are `alert`, `success`, `warning`.
- **`summaryTitle`** - `string` - The title in the summary box.
- **`summaryValue`** - `string` - The secondary text in the summary box.

## Import
```jsx
import { SummaryHeader } from "station-ui";
```
