# Send Header

##### A Header to show information about a send.

## Props
- **`heading`** - <RequiredText /> - `string` - The small text at the top.
- **`label`** - <RequiredText /> - `string` - The main large text in the middle.
- **`subLabel`** - <RequiredText /> - `string` - The secondary text on the bottom.

## Import
```jsx
import { SendHeader } from "station-ui";
```
