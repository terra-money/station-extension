# Section Header

##### A small header to display a label.

## Props
- **`title`** - `string` - The title of the header.
- **`extra`** - `ReactNode` - Adds whatever is passed on the far ride side of the header.
- **`withLine`** - `string` - Adds a line to the header. (Makes the label middle aligned).
- **`indented`** - `string` - Adds a small padding to the left and right of the header.
- **`icon`** - `string` - Adds an icon in front of the label (the prop "title").
- **`className`** - `string` - Adds a class to the header.

## Import
```jsx
import { SectionHeader } from "station-ui";
```
