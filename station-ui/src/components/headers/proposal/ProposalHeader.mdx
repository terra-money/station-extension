# Proposal Header

##### A Header for displaying information about a proposal.

## Props
- **`title`** - <RequiredText /> - `string` - The title of the page (The large text).
- **`metaText`** - <RequiredText /> - `string` - The description of the page (The small text on top).
- **`metaImage`** - <RequiredText /> - `string` - The image of the page (The small image on top).
- **`submittedDate`** - <RequiredText /> - `string` - The date the proposal was submitted.

## Import
```jsx
import { ProposalHeader } from "station-ui";
```
