# Page Header

##### A Header used at the top of a page. Displays the page name and a description.

## Props
- **`title`** - <RequiredText /> - `string` - The title of the page.
- **`description`** - <RequiredText /> - `string` - The description of the page.
- **`icon`** - <RequiredText /> - `ReactNode` - An Icon that correlates with the page.

## Import
```jsx
import { PageHeader } from "station-ui";
```
