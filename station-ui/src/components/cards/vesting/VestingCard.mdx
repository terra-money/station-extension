# VestingCard

##### A Card to display vesting information.

## Props
- **`vestedAmount`** - <RequiredText /> - `string` - The threshold for the ProgressBar.
- **`children`** - `ReactNode` - The content to be displayed on this card.
  - Should be **`<TokenSingleChainListItem />`** for this component.
- **`className`** - `string` - ClassName override for the card.

## Import
```jsx
import { VestingCard } from "station-ui";
```
