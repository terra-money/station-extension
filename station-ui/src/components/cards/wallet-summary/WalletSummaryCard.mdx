# Wallet Summary Card

##### A Card to display Wallet Summary information.

## Props
- **`walletType`** - <RequiredText /> - `string` - To display the wallet type.
- **`walletAddress`** - <RequiredText /> - `string` - The address of the wallet.
- **`walletBalance`** - <RequiredText /> - `string` - The balance of the wallet.
- **`prevTransactions`** - <RequiredText /> - `string` - The 'number' of transactions (as a string).
- **`isActive`** - `boolean` - Set if the card is active.
- **`className`** - `string` - ClassName override for the card.
- **`onClick`** - `() => void` - A function of what to do when this card is clicked.

## Import
```jsx
import { WalletSummaryCard } from "station-ui";
```
