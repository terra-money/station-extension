# SummaryColumn

##### Displays a Title and a Description in a column on a card.

## Props
- **`title`** - <RequiredText /> - `string` - Medium text title.
- **`extra`** - `ReactNode` - Add as a ReactNode item to the right of the title.
- **`description`** - <RequiredText /> - `string` - Description of the card.
- **`className`** - `string` - ClassName override for the card.

## Import
```jsx
import { SummaryColumn } from "station-ui";
```

## Stories

## Default
A card with a title and description.

# SummaryTable

##### Displays rows of data in a table on a card.

## Props
- **`rows`** - <RequiredText /> - `**object below**` - The row data of the card.
  - **`{ label: string; value: string | ReactNode }[]`**
- **`className`** - `string` - ClassName override for the card.

## Import
```jsx
import { SummaryTable } from "station-ui";
```
