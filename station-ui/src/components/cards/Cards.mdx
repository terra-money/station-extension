# Cards
Cards are used to display information in a structured way. Will likely contain other components.

# DelegationCard [<InLineDocsLink />](/docs/components-cards-delegation--docs)

##### This is used to show a wallets individual delegation information.

## Props
- **`validator`** - <RequiredText /> - `**object below**` - Data for the validator to display.
  - `{ name: string; img: string; description: string }`
- **`onClick`** - <RequiredText /> - `() => void` - OnClick function for the card.
- **`className`** - `string` - ClassName override for the card.
- **`children`** - `ReactNode` - Children to display on the card below the validator.
  - Should be `<TokenSingleChainListItem />` for this component.

## Import
```jsx
import { DelegationCard } from "station-ui";
```

# ProposalCard [<InLineDocsLink />](/docs/components-cards-proposal--docs)

##### This is used to show a wallets individual delegation information.

## Props
- **`proposal`** - <RequiredText /> - `**object below**` - Data for the proposal to display.
  - `{ metaText: string; metaImage: string; title: string; submittedDate: string; }`
- **`progressData`** - <RequiredText /> - `ProgressBarData[]` - Data for the `<ProgressBar />`.
  - **`ProgressBarData`** = `{`
    - **`type`**: `'yes' | 'abstain' | 'no' | 'noWithVeto' | 'deposit'`
    - **`percent`**: `string`
  - `}`
- **`threshold`** - <RequiredText /> - `number` - The threshold for the ProgressBar.
- **`progressLabelOverride`** - `string` - Override for the progress label.
- **`className`** - `string` - ClassName override for the card.

## Import
```jsx
import { ProposalCard } from "station-ui";
```

# Wallet Summary Card [<InLineDocsLink />](/docs/components-cards-walletsummary--docs)

##### A Card to display Wallet Summary information.

## Props
- **`walletType`** - <RequiredText /> - `string` - To display the wallet type.
- **`walletAddress`** - <RequiredText /> - `string` - The address of the wallet.
- **`walletBalance`** - <RequiredText /> - `string` - The balance of the wallet.
- **`prevTransactions`** - <RequiredText /> - `string` - The 'number' of transactions (as a string).
- **`isActive`** - `boolean` - Set if the card is active.
- **`className`** - `string` - ClassName override for the card.
- **`onClick`** - `() => void` - A function of what to do when this card is clicked.

```jsx
import { WalletSummaryCard } from "station-ui";
```
