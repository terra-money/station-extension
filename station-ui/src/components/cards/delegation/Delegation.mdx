# DelegationCard

##### This is used to show a wallets individual delegation information.

## Props
- **`validator`** - <RequiredText /> - `**object below**` - Data for the validator to display.
  - `{ name: string; img: string; description: string }`
- **`onClick`** - <RequiredText /> - `() => void` - OnClick function for the card.
- **`className`** - `string` - ClassName override for the card.
- **`children`** - `ReactNode` - Children to display on the card below the validator.
  > - Should be `<TokenSingleChainListItem />` for this component.

## Import
```jsx
import { DelegationCard } from "station-ui";
```
