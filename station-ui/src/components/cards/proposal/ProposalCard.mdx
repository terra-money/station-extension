# ProposalCard

##### This is used to show a wallets individual delegation information.

## Props
- **`proposal`** - <RequiredText /> - `**object below**` - Data for the proposal to display.
  - `{ metaText: string; metaImage: string; title: string; submittedDate: string; }`
- **`progressData`** - <RequiredText /> - `ProgressBarData[]` - Data for the `<ProgressBar />`.
  - **`ProgressBarData`** = `{`
    - **`type`**: `'yes' | 'abstain' | 'no' | 'noWithVeto' | 'deposit'`
    - **`percent`**: `string`
  - `}`
- **`threshold`** - <RequiredText /> - `number` - The threshold for the ProgressBar.
- **`progressLabelOverride`** - `string` - Override for the progress label.
- **`className`** - `string` - ClassName override for the card.

## Import
```jsx
import { ProposalCard } from "station-ui";
```
